- 쇼핑몰 판매자 페이지

 - 0단계 : 준비단꼐
 	1. 라이브러리 복사 ---> web-inf -> lib
 		: ojdbc8.jar, jstl1.2.jar, cos.jar 
 	2. 데이터베이스를 구축 ----> produnt.SQL
 		: 테이블 생성, 시퀀스 생성
 	3. 샘플 데이터 삽입 (5) ---> product.SQL (insert)
 		: 이미지 지정된 위치에 저장
 	4. VO 객체 생성 ---> ProductVO.java
 		: 테이블 기준
 	5. DBCP 셋팅
 	6. DAO 객체 생성 -----------> ProductDAO.java
 		: 싱글턴패턴, 연결,해제 메서드제작
 	7. 모든 페이지 공통으로 적용될 css 제작 --------------------> product.css

-------------------------------------------------------------------
 - 1단계 : 상품 목록을 보여주는 페이지 제작
 	1. index.jsp -> 서블릿 이동                    ----------------> productListServlet
 	2. 서블릿 객체 생성 -----------------------------------> productListServlet
 	3. DAO객체의 상품 전체 정보를 읽어오는 메서드 제작 --------------> 
 	4. 상품 목록을 보여주는 JSP를 제작 ----------------------> productList.jsp
 	
 	
-------------------------------------------------------------------
 - 2단계 : 상품 등록을 처리하는 기능 구현
 	1. 목록페이지에서 등록버튼 생성 ---------------------> productList.jsp -> <a href="productWrite"> 상품 등록 </a>
 	2. 서블릿(doGet) -> 폼페이지로 이동                         -------------> ProductWriteServlet.java -> productWrite.jsp
 	3. 폼페이지(jsp)를 구성 -> 서블릿(doPost)		----->  productWrite.jsp -> ProductWriteServlet.java
 	4. DAO객체에 상품을 Insert하기 위한 메서드			---->  public void insertProduct(ProductVO pVo)
 	5. 서블릿(doPost)에서 목록페이지로 이동하도록               -----> ProductWriteServlet.java
 	 
-------------------------------------------------------------------
 - 3단계 : 상품 상세보기를 처리하는 기능 구현
	1. 목록페이지에서 링크를 생성                                   --------> <a href="productDetail?code=${product.code}"> ${product.name} </a>
	2. 서블릿(doGet)                        -----> ProductDetailServlet.java
	3. DAO 객체에서 상품 하나의 정보를 조회하기 위한 메서드 ----->
	4. JSP에 정보를 넘겨주는 서블릿
	5. JSP제작


--------------------------------------------------------------------
 - 4단계 : 상품 수정을 처리하는 기능 구현
 	1. 목록 및 상세보기에서 링크 생성
 	2. 서블릿(doGet)
 	3. DAO 객체에 상품을 삭제하기 위한 메서드
 	4. alert 명령으로 삭제 확인 메시지 출력
 	5. 목록으로 돌아가기
 
 
-------------------------------------------------------------------
 - 5단계 : 상품 삭제를 처리하는 기능 구현
 	1. 목록 및 상세보기에서 링크 생성
 	2. 서블릿(doGet)
 	3. 수정 폼 페이지로 연결
 	4. JSP 폼페이지 작성
 	5. 서블릿(doPost)
 	6. DAO 객체에 상품을 수정하기 위한 메서드
 	7. 목록으로 돌아가기(OR 상세보기)

-------------------------------------------------------------------
 - 6단계 : 파일을 제어 -> 업로드된 파일중에 불필요한 파일은 제거
